## Installation

1. Install dependencies:
```bash
npm install
```

## Running the Server

Start the server:
```bash
npm start
```

Or use nodemon for auto-restart during development:
```bash
npm run dev
```

The server will run on `http://localhost:3000`

## Usage

1. Open your browser and navigate to `http://localhost:3000`
2. Enter a username (or leave blank for a fun random one!)
3. Start chatting! Try typing `/help` to see available commands

## Fun Commands

- `/joke` - Get a random programming joke
- `/dance` - Show off your dance moves
- `/wave` - Wave hello to everyone
- `/count` - See how many users are online
- `/help` - Show all available commands
